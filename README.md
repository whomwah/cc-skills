# cc-skills

Claude Code skills you can install via:

```sh
npx skills@latest add whomwah/cc-skills/<skill>
```

## Available Skills

| Skill | Description |
|-------|-------------|
| [commit-code](./commit-code/SKILL.md) | Commit staged changes with semantic commit messages |
| [ultraplan-local](./ultraplan-local/PLAN.md) | Produce a thorough implementation plan via multi-agent parallel exploration, critique, and approval |
